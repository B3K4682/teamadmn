# TeamAdmin

TeamAdmin is a web application (admin panel) for any use case. It is built with React.JS and Chakra.

## Installation

1. Clone the repo
   ```sh
   git clone 
   ```
2. Install NPM packages
3. Run the app
   ```sh
   npm start
   ```
4. Build the app
   ```sh
    npm run build
   ```
